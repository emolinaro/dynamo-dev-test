# Text-to-Video

Current Dynamo docs discuss multiple text-to-video paths:

- vLLM-Omni
- SGLang video generation
- FastVideo
- TensorRT-LLM video diffusion

This folder keeps the general text-to-video entry points together. Dedicated notes for FastVideo and TensorRT-LLM also live in their own subfolders under `GenAI/`.
